## Upgrading

Follow these steps to upgrade Janek Unon to the latest version. You back up your data first to avoid issues.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your configurations and databases:
    
    ```bash
    janek-unon backup --output=/path/to/backup
    ```
  </Step>
  <Step title="Update Package" icon="package">
    Install the latest version using your package manager.
  </Step>
  <Step title="Run Migrations" icon="git-pull-request">
    Apply database migrations:
    
    ```bash
    janek-unon migrate
    ```
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your Janek Unon instance and verify functionality.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm" icon="package">
    <CodeGroup tabs="Update,Verify">
      ```bash
      npm update janek-unon
      ```
      ```bash
      npm list janek-unon
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Docker" icon="docker">
    Pull the latest image and redeploy:
    
    ```bash
    docker pull janekunon/latest
    docker-compose up -d
    ```
  </Tab>
  <Tab title="Yarn" icon="package">
    ```bash
    yarn upgrade janek-unon@latest
    ```
  </Tab>
</Tabs>
